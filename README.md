Blinkit Sales Analysis Dashboard
📊 Project Overview

The Blinkit Sales Analysis Dashboard is an interactive Business Intelligence dashboard developed using Microsoft Power BI to analyze Blinkit's sales performance, customer ratings, outlet performance, and product distribution. The dashboard provides meaningful insights into sales trends, outlet characteristics, and product categories, enabling data-driven business decisions.

The dashboard is designed with interactive filters and visualizations, allowing users to explore data dynamically across multiple dimensions such as outlet type, outlet size, item category, and outlet location.

🎯 Project Objective

The main objective of this project is to:

Analyze Blinkit's overall sales performance.
Identify top-performing products and outlet types.
Compare sales across different outlet sizes and locations.
Evaluate customer satisfaction using average ratings.
Provide business insights for improving sales and inventory management.
